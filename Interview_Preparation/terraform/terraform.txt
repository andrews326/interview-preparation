installing terraform::

   get binary by using the following command:

   curl -LO https://releases.hashicorp.com/terraform/1.0.0/terraform_1.0.0_linux_amd64.zip
   unzip terraform_1.0.0_linux_amd64.zip
   sudo mv terraform /usr/local/bin/



 hcl basics
 

   its cotain blocks and arguements
   a block defined within curly braces and its contain set of key value pairs format
    representing the cofiguration data

   a block is a container for multiple arguments


   syntax ::

    resource "resource_type" "resource_name" {
      # configuration options
    }

    resource is a keyword
    resource type is the type of resource being created
    resource name is the unique name for the resource (custom name)

    resource "aws_instance" "example" {
      ami           = "ami-123456"
      instance_type = "t2.micro"
    }

    resource =>> block type
    aws_instance =>> resource type
    example =>> resource name
